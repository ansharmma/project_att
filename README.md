# Student Attendance Pattern Analyzer

## Project Overview
This project analyzes student attendance data using Python.

It helps identify:
- Attendance percentage per student
- Most bunked day of the week
- Overall attendance trends
- Visual representation of data using graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Flask

## Key Features
- Data cleaning and preprocessing
- Attendance trend analysis
- CSV file handling
- Data visualization using bar charts and line graphs

## Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Data visualization
- Backend integration with Flask
